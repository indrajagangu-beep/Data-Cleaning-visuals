# 🧹 Data Cleaning & Reporting Automation using Power BI

## 📌 Project Overview

This project demonstrates how to clean, transform, and automate business reports using **Power BI** and **Power Query**. The dataset contains employee information with intentionally introduced data quality issues such as missing values, duplicate records, inconsistent text formatting, and incorrect data types.

The goal is to automate the data cleaning process and create an interactive dashboard that updates automatically whenever new data is added.

---

## 🎯 Objectives

- Clean and preprocess raw employee data.
- Handle missing values and duplicate records.
- Standardize inconsistent data.
- Build an interactive Power BI dashboard.
- Automate report generation using Power BI Refresh.

## 📂 Dataset Information

The dataset contains **300 employee records**.

### Columns

| Column Name | Description |
|-------------|-------------|
| Employee ID | Unique employee identifier |
| Name | Employee name |
| Department | Employee department |
| City | Employee location |
| Gender | Male/Female |
| Age | Employee age |
| Salary | Monthly salary |
| Joining Date | Date of joining |
| Performance Rating | Employee performance score (1–5) |

### Data Quality Issues Included

- Missing Salary values
- Missing Department values
- Duplicate Employee IDs
- Extra spaces in employee names
- Inconsistent text formatting

These issues are intentionally included to demonstrate the data cleaning process.

## 🚀 How to Run

1. Clone or download this repository.
2. Open the `.pbix` file using **Power BI Desktop**.
3. Load the provided Excel dataset.
4. Click **Refresh** to update the dashboard.
5. Explore the dashboard using the interactive filters.
